# CraftRunner3D
CraftRunner3D est un petit jeu conçu sur Unity 3D basée sur les tutoriels en anglais proposé par brackeys.

## Objectif de jeu
Le jeu n'a pas d'objectif fixe, vous devez simplement esquivez des cubes qui vient faire vous
